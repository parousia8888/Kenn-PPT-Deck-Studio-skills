Decisive keynote CTA for ask / contact / next-step actions.

```jsx
<Button>Request the data room</Button>
<Button variant="ghost">View deck</Button>
<Button variant="link" as="a" href="#">hello@investorsignal.co</Button>
```

- `variant`: `primary` (accent fill) · `ghost` (hairline) · `link`.
- `size`: `sm` · `md` (default) · `lg` for ask slides.
- One primary action per slide. Press state nudges down 1px — no bounce.
