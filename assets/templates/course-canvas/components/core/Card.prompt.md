Raised paper card for concepts, definitions, and grouped content. Optional mono kicker, serif title, and a top accent rule.

```jsx
<Card kicker="Branch B — Parameters" title="The two numbers" accent="sage">
  <p>Mean μ sets the center; standard deviation σ sets the width.</p>
</Card>
```

`accent`: `none` (default), `ox` (oxblood top rule), `sage` (green top rule). Composes with Badge, KeyTerm, Button inside.
