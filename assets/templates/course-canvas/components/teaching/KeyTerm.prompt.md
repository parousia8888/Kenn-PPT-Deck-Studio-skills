Inline emphasis for a key term inside teaching prose.

```jsx
A <KeyTerm>z-score</KeyTerm> rewrites a value as how many <KeyTerm variant="ox">σ</KeyTerm> from the mean.
```

Variants: `highlight` (straw highlighter swipe, default), `underline` (straw rule), `ox` (oxblood ink). Renders a semantic `<mark>`.
