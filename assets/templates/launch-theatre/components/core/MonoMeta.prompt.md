Keynote metadata row — the brand's signature readout of label/value pairs in tracked mono.

```jsx
<MonoMeta items={[
  { label: "Model", value: "LT-9 Aurora" },
  { label: "Version", value: "v2.0", accent: true },
  { label: "Date", value: "06 / 13 / 2026" },
]} />
```

Use on the cover, closing, and benchmark slides. Mark one value `accent` to draw the eye.
