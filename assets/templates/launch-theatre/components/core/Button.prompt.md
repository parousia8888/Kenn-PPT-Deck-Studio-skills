Cinematic CTA button — use for any stage action; reserve `solid` for the one primary action per slide.

```jsx
<Button variant="solid" size="lg">Watch the keynote</Button>
<Button variant="primary">Read the docs</Button>
<Button variant="ghost" size="sm">Skip intro</Button>
```

Variants: `solid` (accent fill, dark text), `primary` (accent outline + glow on hover), `secondary` (neutral outline), `ghost` (text only). Sizes `sm | md | lg`. Pass `href` to render as a link, `icon` / `iconRight` for glyphs.
