A restrained, mono uppercase button for evidence reports — no shadow, near-zero radius.

```jsx
<Button variant="primary">Open dossier</Button>
<Button variant="secondary" size="sm">Export</Button>
<Button variant="ghost" disabled>Locked</Button>
```

Variants: `primary` (International Blue fill), `secondary` (ink outline → ink fill on hover), `ghost` (faint rule). Sizes `sm` / `md`. Use sparingly — reports are documents, not apps.
