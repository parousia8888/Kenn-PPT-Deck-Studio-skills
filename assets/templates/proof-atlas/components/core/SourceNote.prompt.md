The provenance line that accompanies every data point in Proof Atlas. A blue square bullet, then `SOURCE · method · id`.

```jsx
<SourceNote method="internal telemetry" id="SRC-2087" />
<SourceNote method="audit panel" id="n=412" align="right" />
```

Rule: no number ships without one. Pair it as the footer of an `EvidenceCell`, `Metric`, table, or chart.
