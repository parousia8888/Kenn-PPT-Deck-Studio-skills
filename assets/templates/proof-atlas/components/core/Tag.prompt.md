A mono uppercase structural label — section tags, statuses, finding numbers.

```jsx
<Tag>Methodology</Tag>
<Tag variant="accent">Verified</Tag>
<Tag variant="risk">Limitation</Tag>
<Tag variant="solid" index={3}>Finding</Tag>
```

Variants: `default`, `accent` (blue), `solid` (ink fill), `risk` (oxide). Pass `index` for a zero-padded leading number.
